# Docs

Primaire docs-ingang:
- `docs/project/README.md`

Korte routing:
- `docs/project/**` = actieve projectwaarheid
- `docs/project/generated/**` = generated uploadoutput
- `docs/design/generated/**` = generated design-handoff output
- `docs/upload/**` = generated uploadset voor gebruiker; niet gebruiken als canonieke agentbron
- `docs/project/archive/**` en `docs/design/archive/**` = historisch, niet leidend
- `docs/dev/**` = operationele ontwikkelnotities en lokale dumps
- `docs/setup/**` = lokale setup/readiness
