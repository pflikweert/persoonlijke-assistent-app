# Narrative Processing Engine v1 — Persoonlijke Assistent App

## Doel van dit document
Dit document beschrijft hoe ruwe gebruikersinput (tekst/audio) wordt omgezet naar:
- entries  
- dagboekdagen  
- week- en maandreflecties  

Het doel is:
consistent, rustig en betrouwbaar leesbare persoonlijke context genereren zonder interpretatieve AI-gedrag.

## Kernprincipe
De engine structureert en verwoordt — maar interpreteert niet.

## Verwerkingslagen

### Laag 1 — Bron
- tekst
- audio → transcript

### Laag 2 — Entry
Output:
- titel
- opgeschoonde tekst
- korte samenvatting

Regels:
- dicht bij bron
- geen interpretatie

### Laag 3 — Dagboekdag
Output:
- dagsamenvatting
- verhalende dagtekst
- kernpunten

## Schrijfregels — Dagtekst
- IK-vorm
- vloeiend verhaal
- spreektaal → rustig Nederlands
- emoties behouden
- geen nieuwe informatie

## Wat NIET mag
- psychologische analyse
- labels / diagnoses
- therapietaal
- inzichten verzinnen

## Toon
- rustig
- reflectief
- helder
- zonder oordeel

## Dagsamenvatting
- 2–3 zinnen
- feitelijk
- licht reflectief

## Kernpunten
- concreet
- kort
- geen analyse

## Reflecties (week/maand)
Output:
- samenvatting
- gebeurtenissen
- patronen
- reflectiepunten

Regels:
- lichte abstractie
- geen interpretatie

## Fundament
Geen betekenis toevoegen, maar zichtbaar maken.

## Buiten scope
- coaching
- CPTSS
- retrieval
- personalisatie

## Samenvatting
Van ruwe input naar verhalende dag en lichte reflectie, zonder interpretatieve AI.
