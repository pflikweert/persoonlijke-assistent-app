# Gebruik
Alleen voor schermen, routes en UI-structuur.

# Werkwijze
1. Start met scaffold/shared fit check (`docs/dev/ui-assembly-decision-tree.md`).
2. Lees bestaande route- en componentpatronen.
3. Houd wijziging klein en schermgericht.
4. Behoud rustige UI en duidelijke primaire actie.
5. Vermijd nieuwe patronen als bestaande volstaan.
6. Lever alleen noodzakelijke UI-bestanden.

# Niet doen
- Geen schema/migrationwerk.
- Geen server-side OpenAI-flowwerk.
- Geen uitbreidingen buiten gevraagde schermscope.
