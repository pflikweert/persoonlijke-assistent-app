# Gebruik
Alleen voor schermen, routes en UI-structuur.

# Werkwijze
1. Lees bestaande route- en componentpatronen.
2. Houd wijziging klein en schermgericht.
3. Behoud rustige UI en duidelijke primaire actie.
4. Vermijd nieuwe patronen als bestaande volstaan.
5. Lever alleen noodzakelijke UI-bestanden.
